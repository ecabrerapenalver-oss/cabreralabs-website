Privacy Policy — Calendar Heatmap for Power BI

About this visual

Calendar Heatmap (Navigable Calendar Heatmap) is a custom visual for Microsoft Power BI developed and published by Cabrera Labs Inc. (Esteban Cabrera Peñalver). It displays daily values on a navigable monthly calendar with heatmap coloring and cross-filtering support.

What data does the Calendar Heatmap collect?

The Calendar Heatmap visual does not collect, transmit, or store any of your Power BI report data. All calendar rendering, heatmap calculations, and cross-filtering operations happen locally within the Power BI environment.

The only network communication occurs during license validation:

  • License key: When a user enters a license key in the Format panel, the visual sends the key to our server to verify it is valid.
  • Browser fingerprint: An anonymous, non-reversible hash is generated from generic browser properties (user agent, screen size, timezone, language) to count the number of device activations. This fingerprint cannot identify any individual user.

No other data is collected. No report data, no user names, no email addresses, and no usage analytics are transmitted by the visual.

Trial period data

During the 7-day free trial, the visual stores a single timestamp in the browser's localStorage to track when the trial started. This data is stored locally on the user's device and is never transmitted to any server.

Third-party services

The following services are used exclusively for license management and purchase processing:

  • Stripe, Inc. — payment processing during license purchase. See Stripe's privacy policy.
  • Resend, Inc. — transactional email delivery (license key email).
  • Railway Corp. — backend hosting for license validation.
